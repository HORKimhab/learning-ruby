# 📚 Resources to Learn Ruby

A curated list of resources for learning Ruby, from beginner to advanced.

---

## 🌐 Free Online Resources

### Official Documentation
- [Ruby Official Docs](https://www.ruby-lang.org/en/documentation/) — The authoritative reference for Ruby syntax and standard library.
- [Ruby-Doc.org](https://ruby-doc.org/) — Searchable API documentation for Ruby core and standard library.

### Interactive Learning
- [Try Ruby](https://try.ruby-lang.org/) — An official, browser-based Ruby tutorial for absolute beginners.
- [Codecademy – Learn Ruby](https://www.codecademy.com/learn/learn-ruby) — Structured beginner course with hands-on exercises.
- [Exercism – Ruby Track](https://exercism.org/tracks/ruby) — Practice Ruby through mentor-reviewed coding exercises.
- [RubyMonk](https://rubymonk.com/) — Interactive Ruby tutorials from beginner to advanced.

### Written Guides & Tutorials
- [The Odin Project – Ruby](https://www.theodinproject.com/paths/full-stack-ruby-on-rails/courses/ruby) — A comprehensive, project-based free curriculum.
- [Ruby in Twenty Minutes](https://www.ruby-lang.org/en/documentation/quickstart/) — Official quick-start guide for complete beginners.
- [Ruby Style Guide](https://rubystyle.guide/) — Community-driven Ruby coding conventions and best practices.

---

## 📖 Books

### Free / Online
- [Why's (Poignant) Guide to Ruby](https://poignant.guide/) — A quirky, creative introduction to Ruby (legendary in the community).
- [Learn Ruby the Hard Way](https://learnrubythehardway.org/book/) — Exercise-based introduction for beginners.
- [The Well-Grounded Rubyist (excerpts)](https://www.manning.com/books/the-well-grounded-rubyist-third-edition) — One of the most respected Ruby books; covers Ruby deeply.

### Paid Books (Highly Recommended)
- *Programming Ruby 3.2 (The Pickaxe Book)* – Dave Thomas — The definitive Ruby reference.
- *The Well-Grounded Rubyist* – David A. Black & Joseph Leo III — Deep dive into Ruby internals.
- *Eloquent Ruby* – Russ Olsen — Teaches idiomatic, Ruby-style coding.
- *Practical Object-Oriented Design in Ruby (POODR)* – Sandi Metz — Essential for learning OOP in Ruby.

## 🛠️ Practice & Projects

- [LeetCode](https://leetcode.com/) — Solve algorithms in Ruby.
- [Codewars](https://www.codewars.com/?language=ruby) — Gamified coding challenges in Ruby.
- [HackerRank – Ruby](https://www.hackerrank.com/domains/ruby) — Ruby-specific challenges with increasing difficulty.
- [Advent of Code](https://adventofcode.com/) — Annual coding puzzles, great for practicing Ruby.

## 👥 Community & Help

- [Ruby Forum](https://www.ruby-forum.com/) — Official Ruby discussion forum.
- [r/ruby](https://www.reddit.com/r/ruby/) — Active Ruby subreddit.
- [Ruby Discord](https://discord.gg/ruby) — Real-time chat with Ruby developers.
- [Stack Overflow – Ruby](https://stackoverflow.com/questions/tagged/ruby) — Q&A for specific Ruby problems.

---

## 🗺️ Suggested Learning Path

1. **Start here →** [Try Ruby](https://try.ruby-lang.org/) *(30 min)*
2. **Build foundations →** [The Odin Project – Ruby](https://www.theodinproject.com/paths/full-stack-ruby-on-rails/courses/ruby)
3. **Practice daily →** [Exercism](https://exercism.org/tracks/ruby) or [Codewars](https://www.codewars.com/?language=ruby)
4. **Go deeper →** Read *Eloquent Ruby* or *The Well-Grounded Rubyist*
5. **Build projects →** Start a Ruby on Rails app or contribute to open source

---

> 💡 **Tip:** The Ruby community is known for being friendly and welcoming. Don't hesitate to ask questions in forums or Discord!
